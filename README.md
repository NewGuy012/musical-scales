# Made with :heart: with JavaScript + TypeScript + React + Deno

## Musical Scales

Musical scales generates the scale and common chords based on a user given key.

## Running Locally

You need to have Deno v1.28.0 or later installed to run this repo.

Start a dev server:

```
$ deno task dev
```
